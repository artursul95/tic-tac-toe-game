# Игра крестики-нолики
Классическая игра крестики-нолики написанная на языке программирования Python с использованием модуля pygame.

# Как играть
Клонировать репозиторий к себе локально и установить зависимости из файла requirement.txt с помощью команд:<br>  
git clone https://github.com/artursul95/tic-tac-toe-game

pip install -r requirements.txt

Далее просто необходимо запустить исполняемый файл game_gui.py для игры с графическим интерфейсом или файл game_con.py для игры в консольном режиме.

Файл results.txt представляет результаты всех прошедших игр
